G Sheet

Bu məqalədə sənə Python ilə Google Sheets proqramını necə əlaqələndirə biləcəyini göstərəcəm. Məqalədə Google Sheets-ə qoşulmağı, məlumatı istifadə üçün çəkməyi, mövcud məlumatı yeniləməyi sadə nümunələr ilə görə bilərsən.

Python proqramlaşdırma dilinin asanlığına görə bir çox başqa proqramla əlaqələndirilməsi mümkündür ki, bunlardan biri də Google Sheets-dir (bundan sonra “GS” deyəcəm). GS bulud üzərində cədvəl yaratmağa kömək edən bir platformadır. Bu məqalədə həmin platformada olan məlumatları Python ilə necə istifadə edilə biləcəyi haqda danışacam.

Qoşulmaq üçün lazımi işlər#

GS-dən istifadə edə bilmək üçün ilk olaraq mail hesabınla Google Cloud Console-da hesab yaradıb Google Drive API (GD API) və Google Sheets API-ı (GS API) aktivləşdirməyin lazım olacaq. Python ilə platformaya giriş əldə edə bilmək üçün bu gərəkdir.

Formu doldurandan sonra avtomatik olaraq istifadə etdiyin cihaza credentials.json faylı yüklənəcək ki, məhz bu faylda olan məlumatlar ilə GS API-a qoşulmaq üçün istifadə edəcəyik.

Bu faylı əldə edəndən sonra ilkin mərhələ artıq bitir və kod hissəsinə, yəni əyləncəli hissəyə, keçmək olar.

Qeyd: GS-də istifadə etmək istədiyin cədvəli Yüklənən faylda “client_email” hissəsində gördüyün email hesabı ilə paylaşmaq lazımdır. Biraz öncəki addımda sən GS-ə qoşuldunsa, bu addımda artıq spesifik cədvələ qoşulmaq mümkün olur. Əks təqdirdə qoşulma zamanı xəta mesajı ilə qarşılaşacaqsan, çünki GS məhz sənin cədvəldən istifadə etmək istədiyini bilmir.

Python + GS#

Python ilə tanış olanlar yaxşı bilər ki, növbəti mərhələ artıq lazımi kitabxanaların yüklənməsi ilə bağlıdır. Bizim istifadə edəcəyimiz kitabxanalar gspread və oauth2client adlanır.

pip install gspread oauth2client

gspread kitabxanasını GS üzərində əməliyyatlar yerinə yetirmək üçün, oauth2client kitabxanısını isə GS-ə qoşulmaq üçün istifadə edəcəyik.

import gspread
from oauth2client.service_account import ServiceAccountCredentials

scope = ["https://spreadsheets.google.com/feeds", "https://www.googleapis.com/auth/spreadsheets", "https://www.googleapis.com/auth/drive.file", "https://www.googleapis.com/auth/drive"] 

credentials = ServiceAccountCredentials.from_json_keyfile_name("credentials.json", scope)
client = gspread.authorize(credentials)

sheet = client.open("Tut.or.ial").sheet1

# Məlumat gətirmək
find_col = sheet.find('3').col
print(find_col) # Nəticə: 1

find_row = sheet.find('3', in_column=1).row
print(find_row) # Nəticə: 4

sheet.get_all_records()

row = sheet.row_values(1)
print(row) # Nəticə: ['id', 'post_name', 'post_date']

col = sheet.col_values(2)
print(col) # Nəticə: ['post_name', 'G Sheet ', 'Hugo ilə yeni bloq', 'Ödənişsiz vebsayt lazım deyil?', 'Təqdimat "Trick"ləri', 'Siz varsınızsa, təqdimat da var']

cell = sheet.cell(3,2).value  
print(cell) # Nəticə: Hugo ilə yeni bloq 

# Cədvələ məlumat yazmaq
data = [‘sütun_1’, ‘sütun_2’, ‘sütun_3’, 3 , ‘sütun_n’] 
sheet.insert_row(data, 2)

# Cədvəldə olan məlumatı yeniləmək
sheet.update('B3:J3', data)
sheet.update_cell(2,2, "DƏYİŞDİ")

Yuxarıda gördüyün proqram ilə cədvələ qoşulub həmin cədvəldən məlumatları yükləyə bilirik. İndi isə hissə-hissə proqramı izah etməyə keçək.

import gspread
from oauth2client.service_account import ServiceAccountCredentials

Proqramın bu hissəsində bizə lazım olan kitabxanaları (yəni yüklədiyimiz və ya başqa hər hansı kitabxanalar) fayla import edirik. İlk kitabxana aydın olsa da, ikinci üçün qısa izah lazımdı. Bu kitabxanada bizə lazım olan sadəcə ServiceAccountCredentials hissəsidir ki, biz bundan istifadə edərək GS-ə qoşulacağıq.

scope = ["https://spreadsheets.google.com/feeds", "https://www.googleapis.com/auth/spreadsheets", "https://www.googleapis.com/auth/drive.file", "https://www.googleapis.com/auth/drive"] 

credentials = ServiceAccountCredentials.from_json_keyfile_name("credentials.json", scope)
client = gspread.authorize(credentials) # Açardan istifadə edərək API-dan istifadə edirik

Proqramın scope hissəsində biz hansı icazələri istədiyimiz qeyd edirik (bu hissə indiki tutorial üçün vacib deyil, heç özüm də tam bilmirəm orda nə baş verir) və ServiceAccountCredentials-dan istifadə edərək özümüzə açar (izah edəsi olsam, qoşulmaq üçün lazımı məlumatlar olan dəyişən) yaradırıq. Növbəti sətirdə isə artıq bu açardan istifadə edərək GS-ə qoşulmağa nail oluruq.

client dəyişəni artıq sənin GS hesabında olan və yuxarıda qeyd etdiyim kimi client_email email adresi ilə paylaşdığın hər bir cədvəli özündə əks etdirir.

sheet = client.open("Tut.or.ial").sheet1

Bu hissədə artıq istənilən cədvəli (adını axtarışa) və onun içərisində olan səhifəni açırıq.

find_col = sheet.find('3').col # 0 - dəyəri axtarmaq
print(find_col) # Nəticə: 1

find_row = sheet.find('3', in_column=1).row
print(find_row) # Nəticə: 4

sheet.get_all_records() # 1 - bütün dəyərləri gətirmək

row = sheet.row_values(1) # 2 - bir sətiri gətirmək
print(row) # Nəticə: ['id', 'post_name', 'post_date']

col = sheet.col_values(2) # 3 - bir sütunu gətirmək
print(col) # Nəticə: ['post_name', 'G Sheet ', 'Hugo ilə yeni bloq', 'Ödənişsiz vebsayt lazım deyil?', 'Təqdimat "Trick"ləri', 'Siz varsınızsa, təqdimat da var']

cell = sheet.cell(3,2).value  # 4 - bir xananı gətirmək
print(cell) # Nəticə: Hugo ilə yeni bloq

Artıq istədiyin cədvəli təyin edəndən sonra lazımı məlumatları cədvəldən çəkmək olar. Sadə bir nümunələr ilə qeyd edəsi olsam, sən ya hər hansı bir dəyəri cədvədə axtara (#0), ya cədvəldəki bütün dəyərləri (#1), ya sadəcə bir sətirin (#2) və ya bir sütunun dəyərini (#3), ya da bir xananın dəyərini (#4) gətirə bilərsən.

data = [‘sütun_1’, ‘sütun_2’, ‘sütun_3’, 3 , ‘sütun_n’] # Cədvəlin formatına uyğun list
sheet.insert_row(data, 2) # Yeni sətir əlavə etmək

sheet.update_cell(2,2, "DƏYİŞDİ") # Xananın dəyərini dəyişmək

# cədvələ məlumat yazmaq
data = [10, ‘Stephen Zweig’, ‘22-Feb-2019’] # Cədvəlin formatına uyğun list
sheet.insert_row(data, 6)

# Cədvəldə olan məlumatı yeniləmək
updated_data = [10, ‘Heç bloqun yeridir?’, ‘24-Jan-2021’]
sheet.update('A3:C3', updated_data)
sheet.update_cell(2,2, "DƏYİŞDİ")

Proqramın ən son hissəsi isə GS-dəki cədvələ yeni məlumatları əlavə etməklə bağlıdır. Asanlıqla sətir və ya əlavə edə, spesifik xananın dəyərini dəyişə bilərsən.

Tətbiq yerləri#

GS bir məlumat bazası rolu oynadığına görə bir çox yerdə bu ikilini istifadə edə bilərsən və bu hissə məhz sənin kreativliyinə qalıb. Yuxarıda nümünələrdə gördüyün kimi əsasən iki məqsədə istifadə edilə bilər: məlumat çəkilməsi və ya məlumat yazılması. Use your creativity from now on!

Məsələn, mən bunu Quizzendə Google Forms ilə qeydiyyatdan keçən tələbələrin yarışmada iştirak icazəsi olub-olmadığını yoxlamaq üün istifadə edirdim (GS-ə çevirərək), çünki məlumat davamlı olaraq yenilənirdi. Ondan sonra nəticələri də cədvələ imtahan bitən kimi əlavə edirdim.

Daha ətraflı olaraq Gspread kitabxanasının dokumentasiyasından oxuyub sənə lazım ola biləcək funksiyaları öyrənə bilərsən. Video şəklində izah üçün bu linkə keçid edə bilərsən. Mən özüm də bu videodan öyrənmişəm və bu məqaləni onun qeyd etdiklərinə əlavələr edib yazmışam.
